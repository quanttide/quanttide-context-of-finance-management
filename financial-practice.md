# 财务管理专业实践

来自 2026-07-11 迭代复盘。

## 支出刚性弹性化的制度设计

薪酬延期支付的关键不在于"晚点发"，而在于配套完整：

1. 定义触发条件（现金流不足时）
2. 定义优先级（在职优先）
3. 定义补偿标准（10% 年利率）
4. 定义执行保障（周会公开记录、账本登记）

四条缺一不可。只做第一条是应急措施，四条全做才是制度。

## 浮动审批权的阈值

三层权限结构本身不稀奇，阈值定在哪才是经验：

| 金额 | 决定人 | 逻辑 |
|---|---|---|
| 千元以下 | 一线 | 决策成本接近或超过决策收益 |
| 千元到万元 | 合伙人 | 需要判断但不需要最高权限 |
| 万元以上 | 创始人 | 影响现金流，需要全局视角 |

阈值来自真实案例（小几百元软件采购），不是教科书。

## 结构性矛盾识别

"收入弹性、支出刚性"是对现金流紧张的结构化拆解：

- 识别：回款周期不确定，薪资固定
- 制度应对：弹性化支出的机制（延期+利息）
- 工具支撑：`cashflow simulate` 反复演算不同参数下的结果

矛盾识别 → 制度应对 → 工具支撑，三条线完整才算解决了一个财务问题。
